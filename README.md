node-multi-hashing
===============

Cryptocurrency hashing functions for node.js.

Algorithms add
----------
* thriddle      /ThorsRiddle, Veltor/
* lyra2         /Lyra2, Lyra/
* lyra2z330     /Lyra2z8m330, Hexxcoin/
* lyra2z16m330  /Lyra2z16m330, opt Zoincoin/
* timetravel    /Time Travel, Machinecoin/

Algorithms
----------
* quark
* x11
* x13
* nist5
* scrypt
* scryptn
* scryptjane
* keccak
* bcrypt
* skein
* groestl
* blake
* fugue
* qubit
* hefty1
* shavite3
* cryptonight
* boolberry

Usage
-----

Install

* git clone https://github.com/DeffM/node-multi-hashing unomp-multi-hashing
* cd unomp-multi-hashing
* npm link --unsafe-perm
